# CSV-Analytics-Dashboard
 I build a full production-grade HTML website page — a universal CSV analytics dashboard with drag-and-drop upload, auto-detection of columns, and rich interactive charts.
 Here's your DataLens — CSV Analytics Studio as a fully self-contained HTML page. Download it and open it in any browser.
What it does with any CSV you drop in:

Drag & Drop or Browse — accepts any .csv, .tsv, or .txt file with headers
Overview tab — auto-detected metric cards, histogram of first numeric column, top category breakdown, and column range comparisons
Charts tab — pick any column + chart type (bar, line, pie, doughnut, radar) + optional grouping column; plus a gallery of mini-distributions for every numeric column
Statistics tab — full descriptive stats (mean, median, std, quartiles, nulls) for numeric columns, and top-value breakdowns with visual bars for categorical columns
Correlation tab — color-coded Pearson correlation heatmap, correlation bar chart, and ranked top correlated pairs
Scatter tab — X/Y axis selector with optional color-by-category grouping
Data Table — searchable, sortable by any column, paginated (20/50/100 rows per page)

Click "Load football players sample dataset" on the page to see it in action immediately without uploading anything.
